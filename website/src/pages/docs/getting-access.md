# Getting access

TimeCheese has no self-serve sign-up. Accounts are created by the administrator directly in
Supabase.

## How to get access

1. Ask your admin to create a user with your work email.
2. You'll receive credentials, or a first-login OTP email to set your own password.
3. Sign in in the app — that's it.

## FAQ

**Why no sign-up?**

It's an internal tool — provisioning is deliberate.
